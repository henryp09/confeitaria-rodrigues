 Confeitaria Rodrigues

Aplicação web desenvolvida com React, TypeScript e Node.js, criada para apresentar uma confeitaria moderna com navegação simples, design elegante e foco na experiência do usuário.

 Sobre o projeto

O Confeitaria Rodrigues é um site criado para representar uma confeitaria real, trazendo:

página inicial com visual atrativo

seção de produtos

interface simples e intuitiva

navegação entre páginas

tipografia e cores personalizadas

Mesmo sendo construído com ferramentas modernas, o projeto mantém uma estrutura agradável e fácil de entender.
